About This Project

Ranking Stories is a small collection of animated data visualizations designed for education, storytelling, and short-form video production. Each page focuses on a clear ranking question and presents long-term trends in a visually engaging format.

Some visualizations use approximate or interpolated values for smooth animation and storytelling. Source notes are provided on individual pages.
